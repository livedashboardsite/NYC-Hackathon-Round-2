## 🏆 **Problem Statement**
### CREATIVE-01 — Make Boring Impossible

> *"Design and build a product or experience that people genuinely enjoy using and naturally want to share with others."*

Our digital lives are cluttered with notifications, and clearing them is a mindless, boring chore. TidyQuest answers the prompt by completely flipping this script: we gamified the exact UI of a notification center and made dismissing them intensely satisfying using visual feedback, streaks, and shareable scores.

## ✨ **Features**

<table align="center">
<tr>
<td width="50%">

### 🎯 **Satisfying Mechanics**
- Click, drag, or use keyboard inputs
- Canvas-based particle explosions
- Screen shake and synthesized audio
- Every action is designed to feel GOOD

### 🔥 **Streak System**
- Fast dismissals build bigger combos
- Visual color shift as your streak heats up
- "Streak broken" warnings keep you on edge

</td>
<td width="50%">

### 📊 **Real-Time Feedback**
- Live score counter
- Dynamic timer tracks your speed
- Progress ring shows remaining targets

### 📸 **Shareable Recap**
- Generates a "Tidier than X%" metric
- One-click download as PNG
- Native sharing to social media

## 🚀 **Installation & Setup**

Because TidyQuest is built entirely with Vanilla web technologies, setup takes less than 10 seconds.

```bash
# 1. Clone the repository
git clone [https://github.com/](https://github.com/)<your-username>/tidyquest.git

# 2. Navigate to the directory
cd tidyquest

# 3. Open in your browser!
# You can just double-click index.html, or use a local server:
npx serve .
